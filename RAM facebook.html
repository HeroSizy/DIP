Object Detection

Object Detection is achieved by using an open-source software(Yolo).The basic theory behind is to apply a single neural network to the full image.Then the network divides the image into regions and predicts bounding boxes and probabilities for each region.It has several advantages compare to convolution network. It looks at the whole image at test time so its predictions are informed by global context in the image. It also makes predictions with a single network evaluation unlike systems like R-CNN which require thousands for a single image. This makes it extremely fast, more than 1000x faster than R-CNN and 100x faster than Fast R-CNN. See our paper for more details on the full system.

Road Condition Analysis

The theory of Road Condition Analysis is based on object detection. The program monitors real-time traffic and output number of cars at same time. By comparing the result with the previous observation,the team manage to anaylsis the road conditon and output a warning signal. Drivers can refer to this result and make decisions to choose their travelling path accordingly. It will save a significant amount of time on transportation.

Unique Function

Besides the traffic analysis, the program also provides the function to detect other vehicles on the road. In some countries like China, truck is only allowed to go on Expressway during certain time period. The program is able to detect the truck and send out a warning alarm to tell traffic police officer that a truck has been detected on the Expressway. Certain precautions maybe carry out by the police. Further implementation of snap shoot of the truck's plate is also available.With object detection technology, there are more innovative applications can be created.
